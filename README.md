# Cyber Slavery Arrest Map

Interactive visualization of cyber slavery arrests across Indian states and union territories.

## About

This map displays arrest data related to cyber slavery cases across India. Hover over states to view details.

## Usage

- View the map to see arrest counts by state
- Color gradient indicates relative arrest numbers (green = low, red = high)
- Hover over any state to see details

## Data

Data is updated periodically by authorized administrators.
